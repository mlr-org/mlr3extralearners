# Polynomial GLM Learner

Polynomial GLM. Calls
[`polyreg::polyFit()`](https://rdrr.io/pkg/polyreg/man/polyFit.html)
from [polyreg](https://CRAN.R-project.org/package=polyreg).

## Initial parameter values

- `deg`: We have set this to 2, pretty arbitrarily.

- `noisy`: We have set this to FALSE, to get no output on the console.

## Dictionary

This [Learner](https://mlr3.mlr-org.com/reference/Learner.html) can be
instantiated via
[lrn()](https://mlr3.mlr-org.com/reference/mlr_sugar.html):

    lrn("classif.polyFit")

## Meta Information

- Task type: “classif”

- Predict Types: “response”

- Feature Types: “numeric”, “factor”
